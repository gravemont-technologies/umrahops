# RPA Scripts

This directory contains scripts for Robotic Process Automation (RPA) with the Nusuk portal.

## Usage

1.  **Do NOT run on Replit**: These scripts require headful browsers.
2.  **Local Run**:
    ```bash
    npx tsx nusuk_rpa_stub.ts
    ```

## Implementation Guide

Modify `nusuk_rpa_stub.ts` to implement the actual selectors for:
- Login
- Group creation
- Traveler data entry
- Visa status polling
